Модули 1–3 — Банкетам.Нет

Я разработала веб-портал для бронирования банкетных помещений
Стек:React, Vite, Tailwind CSS, Node.js, Express, PostgreSQL, Sequelize, JWT.

Что реализовала:
- регистрация и вход пользователей;
- подача заявок на бронирование с проверкой полей;
- личный кабинет и список своих заявок;
- панель администратора (статусы, фильтры, сортировка, поиск);
- слайдер на главной, блок контактов, блок качества со значками;
- фотографии залов из папки `Модули/client/public/assets/`;
- ER-диаграмма и SQL-схема базы данных.

Запуск

Из корня репозитория:

```bash
npm run install:all
npm run dev
```

Или из папки `Модули`:

```bash
cd Модули
npm run install:all
npm run dev
```

Перед первым запуском API скопируйте `Модули/server/.env.example` в `Модули/server/.env` (при `install:all` файл создаётся автоматически, если его ещё нет).

| Сервис | Адрес |
|--------|--------|
| Сайт | http://localhost:3000 |
| API | http://localhost:5000 |
| Админ | логин **Admin26**, пароль **Demo20** |

Если порты заняты:

```bash
cd Модули
npm run free-ports
npm run dev
```
